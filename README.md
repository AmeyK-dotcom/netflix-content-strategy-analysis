# 🎬 Netflix Content Strategy & Performance Analysis

A data analytics project focused on understanding Netflix content trends, performance patterns, and deep-dive analysis of series-level behavior using SQL and Tableau.

---

## 📊 Project Overview

This project analyzes 6,000+ Netflix titles to uncover:

- Content distribution across Movies vs TV Shows  
- Growth trends in Netflix catalog over time  
- Relationship between content quality (IMDb rating) and popularity  
- Deep-dive episode-level analysis of *Peaky Blinders*  

The project is structured into two dashboards for clarity:
1. Executive Overview  
2. Performance & Deep Dive

## 🧠 Key Insights

- Movies dominate Netflix content (~70%+), but TV Shows show deeper engagement  
- Weak correlation between IMDb ratings and popularity → not all high-rated content is widely watched  
- High-impact content identified using performance vs popularity segmentation  
- *Peaky Blinders* shows strong season finales and consistent high ratings across seasons
